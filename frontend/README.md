# Doctor Appointment System - Frontend

## Project Description
The frontend component of the Doctor Appointment System (DAS), an intuitive UI for patients to book appointments and doctors to manage schedules and patient reports.

## Tech Stack
- **React.js 18**
- **React Router**
- **Axios**
- **Custom CSS**

## How to Run Locally

1. Navigate to the `frontend/frontend-ui` directory.
2. Install the necessary NPM packages:
   ```bash
   npm install
   ```
3. Set the required `.env` file referencing the locally running backend.
4. Start the development server:
   ```bash
   npm start
   ```
5. The frontend will be available at `http://localhost:3000`.

## Environment Variables Needed

Create a `.env` file in the `frontend-ui` directory with the following configuration:
```env
REACT_APP_API_URL=http://localhost:8080
```
*(When deploying, set this to the backend API URL)*
